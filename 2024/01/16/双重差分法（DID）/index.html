双重差分法（Difference-in-Difference Estimation）

一、双重差分法能解决什么问题？

1.双重差分法是指分析某处理行为的效应

1.1 什么是处理行为

政策、自然冲击、特定干预等，会产生影响的因素

能用"是"或者"不是"回答是否接受处理行为

1.2 对什么因素的效应（产生效应的对象）

效应对象：研究问题所关注的一个或者多个结果

二、什么样的数据可以使用双重差分法

1.DID采用观测数据

数据可以分为观测数据和实验数据

观测数据（observational data）:观测和收集真实发生的数据，如：股价、GDP、死亡率等

实验数据（experimental data）：通过精准科学设计的实验产生的数据，如：中学实验室中产生的物理、化学实验数据

DID是将观测数据通过两次差分来模拟实验数据

2.数据需要包含处理组和非处理组

处理组：接受了处理行为的个体

非处理组：没接受处理行为的个体

3.样本数据的时间维度要包含处理前和处理后

处理组 非处理组
处理前 有数据 有数据
处理后 有数据 有数据

三、传统双重差分法的原理

处理行为发生在一个时间点：传统双重差分法、经典双重差分法、两期双重差分法

处理行为发生在不同时间点：多期双重差分法、多时点双重差分法、渐进双重差分法

倾向得分匹配是通过人为构造一个对照组，但在实际问题中，找到这样的对照组有困难，因此可以使用双重差分法。

前提课程：学习倾向得分匹配

一、在传统的回归中，实验组和对照组可能会存在选择偏差（selection bias）

如何解决选择偏差？比如可以使用增加更多的控制变量、工具变量IV、外生事件法等，但这些都是建立在回归基础上。

而倾向得分匹配（propensity score matching PSM）也是为了解决选择偏差，但其和回归模型在探索因果的本质上思路是不一样的

四、回归模型中的双重差分法

回归模型：

\[Y_{it} = \beta_{0} + \beta_{1}*T + \beta_{2}*D + \beta_{3}*D*T + \varepsilon\]

\(Y_{it}\)是因变量，表示处理行为的效用对象；\(T\)是时间变量，0-1变量，处理行为发生之前取0，处理行为发生之后取1；\(D\)是处理组变量，0-1变量，非处理组取0，处理组取1；\(D*T\)是一个交互项，只有当是处理组且发生在处理行为之后取1，其余都取0，对应系数为研究的"处理行为的效果"。

则处理组（\(D = 1\)）在处理之前（\(T = 0\)）\(Y_{it} = \beta_{0} + \beta_{2} + \varepsilon\)

处理组（\(D = 1\)）在处理之后（\(T = 1\)）\(Y_{it} = \beta_{0} + \beta_{1} + \beta_{2} + \beta_{3} + \varepsilon\)

非处理组（\(D = 0\)）在处理之后（\(T = 1\)）\(Y_{it} = \beta_{0} + \beta_{1} + \varepsilon\)

非处理组（\(D = 0\)）在处理之前（\(T = 0\)）\(Y_{it} = \beta_{0} + \varepsilon\)

又由于我们不是对一个个体进行研究，而是对样本集进行研究，即取全部样本的平均值\(E\left( Y_{it} \right)\)，所以残差\(\varepsilon\)将全部抵消

处理组 非处理组
处理前 \[\beta_{0} + \beta_{2}\] \[\beta_{0}\]
处理后 \[\beta_{0} + \beta_{1} + \beta_{2} + \beta_{3}\] \[\beta_{0} + \beta_{1}\]

那么，由双重差分法的原理可得，最终处理效应=

\(\left\lbrack \left. （\beta_{0} + \beta_{1} + \beta_{2} + \beta_{3} \right.） - \left. （\beta_{0} + \beta_{2} \right.） \right\rbrack - \lbrack\left( \beta_{0} + \beta_{1} \right) - (\beta_{0})\rbrack\)=\(\beta_{3}\)

因此，\(\beta_{3}\)就是我们想要探究的"处理行为的效果"。

问题一：为什么要用回归模型呢？

问：按照上述双重差分法的原理，我们可以不用回归模型去求解"处理行为的效果"，那么为什么要用回归模型呢？

答：回归可以添加控制变量。

控制变量可以分为影响\(Y_{it}\)的非时间控制变量和时间控制变量。

非时间控制变量

完美多重共线性：回归模型中有一个变量能被另外一个或者多个变量线性代替。

例子：探讨催更对更新频率的影响，已知up主坐在城市为北京、上海、广州，并且所有在上海的up主都被催更，北京、广州的up主都没有被催更。则构建如下模型：

\[Y_{it} =\]

\[\beta_{0} + \beta_{1}*T + \beta_{2}*D + \beta_{3}*D*T + \beta_{4}*{上海}_{it} + \beta_{5}*{北京}_{it} + \beta_{6}*{广州}_{it} + \varepsilon\]

其中存在两处完美多重共线性：

①处理组变量\(D\)与变量\({上海}_{it}\)存在共线性，\(D = {上海}_{it}\)。解决方法：去除\(\beta_{2}*D\)。

②\({上海}_{it}\)、\({北京}_{it}\)、\({广州}_{it}\)三个变量间存在多重共线性，\({上海}_{it} = 1 - {北京}_{it} - {广州}_{it}\)，解决方法：去除\({上海}_{it}\)、\({北京}_{it}\)、\({广州}_{it}\)中的任意一个，或者去掉常数项\(\beta_{0}\)，这两种操作方法对于我们所探究的交互项的系数（\(\beta_{3}\)）是没用影响的，但对于城市变量的系数是有影响的。

时间控制变量

影响结果\(Y_{it}\)的时间控制变量。基本的DID回归模型只有两个时间点（处理前和处理后），并通过时间变量\(T\)进行表示。但大多数的数据中是不止有两个时间点，并且不同的时间点对于处理效应\(Y_{it}\)可能存在影响，因此也需要将不同时间点加入到回归模型中去。但是注意，这不是多期DID，其处理行为仍然是只发生在同一个时间点。

例子：探讨催更对更新频率的影响，加入1-6月变量，处理行为发生在3月。

参照城市变量的添加方法，添加一组时间0-1变量，即将

\[Y_{it} = \beta_{0} + \beta_{1}*T + \beta_{2}*D + \beta_{3}*D*T + \varepsilon\]

替换成：

\[Y_{it} =\]

\[\beta （{1月}_{it} + {2月}_{it} + {3月}_{it} + {4月}_{it} + {5月}_{it} + {6月}_{it}） + \beta_{2}*D + \beta_{3}*D*T + \varepsilon\]

其中\(\beta\)是一个系数矩阵。

但这种替换存在一个前提假设：处理行为在不同时间段产生的影响都是一样的。但是，这与实际情况一般是不符合的。例如：催更之后，随着时间的推移，会越来越懈怠，也就是催更的处理效用随时间越来越小。

因此，对上面的变换还想要进行改进，将交互项中的时间变量\(T\)进行展开：

\[Y_{it} =\]

\[\beta\left. （{1月}_{it} + {2月}_{it} + {3月}_{it} + {4月}_{it} + {5月}_{it} + {6月}_{it} \right.） + \beta_{2}*D +\]

\[\beta_{3}*D*{1月}_{it} + \beta_{4}*D*{2月}_{it} + \beta_{5}*D*{3月}_{it} + {\beta_{6}*D*4月}_{it} + {\beta_{7}*D*5月}_{it} + {\beta_{8}*D*6月}_{it} + \varepsilon\]

但是，这个模型中仍然存在多重共线性：

\[D = D*（{1月}_{it} + {2月}_{it} + {3月}_{it} + {4月}_{it} + {5月}_{it} + {6月}_{it}）\]

当左式\(D = 0\)时，右式也为0，当左式\(D = 1\)时，右式也为1*1=1，因为\({1月}_{it}、{2月}_{it}、{3月}_{it}、{4月}_{it}、{5月}_{it}、{6月}_{it}\)中必定有一个为0，其余为1。所以我们需要去掉一个时间点，一般我们去掉处理发生的时间点（因为处理发生时间点的系数不好解释），即：

\[Y_{it} =\]

\[\beta\left. （{1月}_{it} + {2月}_{it} + {3月}_{it} + {4月}_{it} + {5月}_{it} + {6月}_{it} \right.） + \beta_{2}*D +\]

\[\beta_{3}*D*{1月}_{it} + \beta_{4}*D*{2月}_{it} + {\beta_{6}*D*4月}_{it} + {\beta_{7}*D*5月}_{it}\]

\[+ {\beta_{8}*D*6月}_{it} + \varepsilon\]

其中\(\beta_{3}\)就代表了处理行为在1月份对于处理组的影响，但在1月份处理行为还没有发生（3月份），因此\(\beta_{3}\)应该在经济意义和统计意义上都是不显著的（系数非常小且不显著）。

五、多期双重差分法1

多期双重差分法：处理行为发生在不同时间点，即处理组个体发生处理行为的时间点都是不一样的。

处理行为一定是不可逆的（政策的实施不存在撤销，开通的高铁将一直运营而不关闭，接受更高教育的高管将一直都是）

山东吃货组在3月份去吃了，则3、4、5、6月份的处理行为变量都为1；外省吃货在5月份去吃了，则5、6月份的处理行为变量都为1；非吃货组没有去过，所以处理行为变量为0。

因此对于传统DID回归模型：

\[Y_{it} = \beta_{0} + \beta_{1}*T + \beta_{2}*D + \beta_{3}*D*T + \varepsilon\]

其中时间变量\(T\)可以直接用一组时间的0-1变量替代，将处理变量\(D\)替换成一组组别的0-1变量，将\(D*T\)看着一个整体用一个指示变量进行替代即上图中的"淄博吃烧烤"。因此，原回归模型改写为：

\[Y_{it} = \beta_{0} + \beta_{1}*{1月}_{i} + \beta_{2}*{2月}_{i}{+ \beta}_{3}*{3月}_{i}{+ \beta}_{4}*{4月}_{i}{+ \beta}_{5}*{5月}_{i}{+ \beta}_{6}*{6月}_{i}\]

\[+ \beta_{6}*{山东}_{t} + \beta_{7}*{外省}_{t} + \beta_{8}*{非吃货}_{t}\]

\[+ \beta_{8}*D_{it} + \varepsilon\]

写成更一般的形式：

\[Y_{it} = \beta_{0} + \beta_{1}*\lambda_{t} + \beta_{2}*\gamma_{i} + \beta_{3}*D_{it} + \varepsilon\]

其中\(\lambda_{t}\)是一组时间的0-1变量，即时间固定效应；\(\gamma_{i}\)是一组组别的0-1变量，即个体固定相应，\(\beta_{3}\)是我们所关心的多期双重差分法中处理效应。所以多期DID也叫做双向固定效应模型（Two-way fixed-effects diff-in-diff，TWFEDD）。

但是，从2020年以后，很多学者发现这种双向固定效应模型存在错误！

六、多期双重差分法2

TWFEDD出现错误的原因

《Difference-in-difference with variation in treatment timing》1

还是针对上一节的例子。下图中的三条虚线分别代表山东美食爱好者（3月吃烧烤）、外省美食爱好者（5月吃烧烤）、非美食爱好者（不去吃烧烤）。

并这张图拆解为我们熟悉的传统双重差分法，如下面四张图所示。

对四组传统DID分别计算处理效应，依次为\(\beta_{3}^{DD1}\)、\(\beta_{3}^{DD2}\)、\(\beta_{3}^{DD3}\)、\(\beta_{3}^{DD4}\)，但是对于第四张图，作为对照组的山东省美食爱好者组（上面一根虚线）在3月份已经接受了处理，因此对\(\beta_{3}^{DD4}\)存疑。

使用这样的计算方法，则对于

\[Y_{it} = \beta_{0} + \beta_{1}*\lambda_{t} + \beta_{2}*\gamma_{i} + \beta_{3}*D_{it} + \varepsilon\]

有：

\[\beta_{3} = s_{1}*\beta_{3}^{DD1} + s_{2}*\beta_{3}^{DD2} + s_{3}*\beta_{3}^{DD3} + s_{4}*\beta_{3}^{DD4}\]

其中\(s_{1}\)、\(s_{2}\)、\(s_{3}\)、\(s_{4}\)是一组权重。\(s\)权重受每组样本量、分组比较时间长短、子样本的波动因素影响。

再回到对于第四组处理效果\(\beta_{3}^{DD4}\)，这是存在问题的。在传统双重差分法中，对照组在处理行为发生前后的差值应只受时间因素影响，但如果使用已经接受过处理行为的组别作为对照组，那么其前后差值不仅受到了时间因素影响，还受到了处理行为因素的影响，而处理行为效果随时间一般是要发生变化的。举例：

吃烧烤这个处理行为的效果是第一个月增加2kg，第二个月增高2kg，第三个月恢复正常水平。4-6月由于时间因素，体重每月会增加0.5kg。则：

体重 4月 5月 6月
山东美食爱好者 A+2kg A +1kg+0.5kg A +0kg+1kg
外省美食爱好者 B B +0.5kg B +2kg+1kg
非吃货 C C +0.5kg C +1kg

那么，将山东组和外省组进行双重差分，以山东组为对照组，则外省组在5月吃烧烤后第一个月带来的处理效应是

[(B +2kg+1kg)-(A +0kg+1kg)]-[B-(A+2kg)]=4kg

这与题目中的假设"吃烧烤这个处理行为的效果是第一个月增加2kg"相矛盾，其原因在于山东组已经接受了处理行为，除了随时间发生变化外，还存在一个处理行为发生后处理效果变化的值。

因此\(\beta_{3}^{DD4}\)是错误的，那么\(\beta_{3}\)就是错误的，所以双向固定效应模型进行多期双重差分也是有问题的。

什么时候可以/不可以使用双向固定效应？

如果处理行为的效果是一个恒定不变的数值，则可以使用双向固定效应，相反则不行。

七、多期双重差分法3

本节处理多期双重差分法的方法来自于《Difference-in-Differences with multiple time periods》2（stata中有CSDID软件包）

中心思想：将整体数据拆分成小模块，每一个小模块内做传统双重差分法（canonical Diff-in-Diff），将小模块求出的处理行为效果进行加权平均。

1、将整体数据拆分成小模块

这就是以尚未处理组作为对照组，但其中也存在三个问题：

①随着时间的推移，对照组个体数量会越来越少。以\(G_{5}\)组为例，处理行为发生当期有\(G_{6}\)、\(G_{7}\)、\(G_{8}\)、\(G_{9}\)组作为对照组，发生后一期有\(G_{7}\)、\(G_{8}\)、\(G_{9}\)组作为对照组，发生后二期有\(G_{8}\)、\(G_{9}\)组作为对照组，发生后三期有\(G_{9}\)组作为对照组。

②检验期间受最后一个处理组别的处理时间限制。

③无法针对最后一个处理组别进行小模块构造。

因此，究竟是选择C组（从未发生处理行为的组别）作为对照组还是暂未发生处理行为的组别作为对照组需要根据数据进行判断。考虑两方面的内容：

①C组的个体是否与其他处理组的个体存在很大差别；

②最后一个处理组别是否在很早就完成了处理行为，以及处理行为发生较晚的组别中包含了多少个体。

2.在小模块内做传统双重差分法（canonical Diff-in-Diff）

假设1：任何组别的处理行为，都不发生在第一期，且处理行为不可逆；

假设2：每个个体（是个体\(i\)而非组别\(G\)）都是独立同分布的；

假设3：对未来处理行为预期有限（也就是处理行为发生之前的时间点不会收到处理行为的影响，即没有对处理行为的预期。但在上述论文中Callaway B, Sant'Anna教授是允许有一定的预期的，但为了方便学习，我们在这里假设没有预期，即处理行为不会对处理行为发生时间点之前有影响）。

假设4、5：平行趋势假设

假设4：如果没有处理行为发生，处理组在处理行为发生当期或者之后的时间点t与时间t-1之间的拜年话，与对照组C组在相同时间点的变化相等。

假设5：如果没有处理行为发生，处理组在处理行为发生当期或者之后的时间点t与时间t-1之间的拜年话，与暂未发生处理行为组（非处理组）在相同时间点的变化相等。
